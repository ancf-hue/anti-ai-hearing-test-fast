# 🟢 «Анти ИИ Проверка слуха» (быстрая тренировка памяти)

**Быстрая версия** — тренировка слуха, памяти и скорости реакции. Цифры произносятся в ускоренном темпе.

---

## ⚡ Отличия от медленной версии

| Параметр | Медленная (🔴) | Быстрая (🟢) |
|----------|---------------|---------------|
| Скорость цифр | 0.3 (очень медленно) | **1.5 (быстро)** |
| Пауза между цифрами | 800 мс | **200 мс** |
| Цветовая схема | Красный #FF0000 | **Зелёный #00FF00** |
| Сложность | Низкая (для начинающих) | **Высокая (для опытных)** |

---

## 🎮 Как играть

1. **Выберите уровень сложности:**
   - `1 · тишина` — чистые цифры
   - `2 · шум средний` — фоновый шум
   - `3 · шум громкий + ИИ` — искажённые звуки + детектив

2. **Нажмите 🔊** — цифры произнесутся **быстро**

3. **Введите цифры** по порядку (кнопки 0-9)

4. **На 3 уровне** определите, какая цифра была искажена ИИ

5. **Нажмите ✅** и получите очки

---
